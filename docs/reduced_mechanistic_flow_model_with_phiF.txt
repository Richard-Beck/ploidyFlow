Reduced mechanistic flow-cytometry binding model with explicit fixation accessibility factor

Purpose
This note gives a derivation of a reduced mechanistic dye-depletion model for DNA peak shifting in flow cytometry, modified to include an explicit tumor-side fixation accessibility factor φ_F. The role of φ_F is to represent a common multiplicative effect of fixation on the effective accessibility of tumor DNA to the dye, while allowing the CEN spike to remain on its own scale.

The key motivation is that DNA content may still determine the baseline tumor binding burden, but fixation can change the accessible fraction of that burden. If the spike/control material was not fixed identically to the tumor material, this factor should not be silently absorbed away.

------------------------------------------------------------------
1. Starting point
------------------------------------------------------------------

For sample s and peak class j,

  F_js = a_j * B_j * (K_j * c_s) / (1 + K_j * c_s)

where

  F_js  = fluorescence of class j in sample s
  a_j   = instrument / fluorescence conversion factor
  B_j   = accessible binding sites per particle of class j
  K_j   = effective association-like binding parameter
  c_s   = free dye concentration in sample s at equilibrium

Total dye conservation in sample s is

  D_0 = c_s + sum_j [ n_js * B_j * (K_j * c_s) / (1 + K_j * c_s) ]

where

  D_0  = total dye added to the tube
  n_js = number of particles of class j in sample s

This is the full mechanistic starting point.

------------------------------------------------------------------
2. Safe reparameterizations, with explicit fixation factor
------------------------------------------------------------------

We now introduce an explicit tumor-side fixation accessibility factor

  φ_F > 0

interpreted as a common multiplicative modifier of tumor accessible binding sites after fixation.

We write tumor accessible binding sites as

  B_m = φ_F * Btilde_m

for tumor classes m, where Btilde_m is the corresponding baseline accessible-site quantity before applying the common fixation accessibility factor.

For the CEN class C, we leave B_C as its own parameter.

CEN reparameterization:

  M_C := a_C * B_C * K_C

Then

  F_Cs = (M_C * c_s) / (1 + K_C * c_s)

Tumor reparameterization:

For tumor peak m,

  F_ms = a_m * (φ_F * Btilde_m) * (K_m * c_s) / (1 + K_m * c_s)

Define

  Mtilde_m := a_m * Btilde_m * K_m

Then

  F_ms = φ_F * (Mtilde_m * c_s) / (1 + K_m * c_s)

Tumor:CEN ratio:

For a tumor peak m relative to the CEN peak C,

  F_ms / F_Cs = φ_F * [(a_m * Btilde_m * K_m) / (a_C * B_C * K_C)] * [(1 + K_C * c_s) / (1 + K_m * c_s)]

Define the baseline asymptotic high-dye-limit ratio, excluding the common fixation factor,

  R*_m,inf := (a_m * Btilde_m * K_m) / (a_C * B_C * K_C)

Then

  F_ms / F_Cs = φ_F * R*_m,inf * (1 + K_C * c_s) / (1 + K_m * c_s)

Interpretation:
  R*_m,inf is the uncompressed tumor:CEN ratio in the high-dye limit, excluding the common tumor-side fixation accessibility factor.
  φ_F is kept explicit so that fixation effects are not absorbed into the ratio parameter.

------------------------------------------------------------------
3. Structural Assumption A: common effective tumor binding curve
------------------------------------------------------------------

Assumption A:

  K_m = K_T   for all tumor classes m.

This does NOT mean all tumor peaks have the same fluorescence. It means they share the same depletion response curve; they still differ through Btilde_m, Mtilde_m, and therefore R*_m,inf.

Under this assumption,

  F_ms / F_Cs = φ_F * R*_m,inf * (1 + K_C * c_s) / (1 + K_T * c_s)

Define the sample-specific distortion factor

  H_s := (1 + K_C * c_s) / (1 + K_T * c_s)

Then

  F_ms / F_Cs = φ_F * R*_m,inf * H_s

Interpretation:
  All tumor peaks share the same sample-specific multiplicative distortion relative to CEN, but they are additionally rescaled by the common fixation accessibility factor φ_F.

Special case:
  If K_T = K_C, then H_s = 1 and tumor:CEN ratios are constant across load, but still satisfy

    F_ms / F_Cs = φ_F * R*_m,inf

  So even if depletion is purely common-mode, fixation can still shift tumor:CEN spacing relative to CEN.

------------------------------------------------------------------
4. Structural Assumption B: tumor binding burden scales with DNA content, modulated by fixation
------------------------------------------------------------------

Starting from dye conservation and applying Assumption A,

  D_0 = c_s
        + n_C,s * B_C * (K_C * c_s) / (1 + K_C * c_s)
        + [sum_m n_m,s * B_m] * (K_T * c_s) / (1 + K_T * c_s)

Define the total effective tumor binding-site burden

  L_T,s := sum_m [ n_m,s * B_m ]

Assumption B:
  Baseline accessible sites per tumor particle are proportional to DNA content:

    Btilde_m = λ * d_m

where d_m is DNA content in diploid-genome units, for example

  d = 2 for 2N G1
  d = 3 for diploid S-phase average
  d = 4 for 2N G2 or 4N G1
  d = 6 for 4N S-phase average
  d = 8 for 4N G2 / 8N-class peak, depending on labeling convention

With explicit fixation accessibility,

  B_m = φ_F * Btilde_m = φ_F * λ * d_m

Define the weighted tumor DNA-load proxy

  W_s := sum_m [ n_m,s * d_m ]

Then

  L_T,s = sum_m [ n_m,s * (φ_F * λ * d_m) ]
        = φ_F * λ * W_s

If the CEN spike is constant across samples, n_C,s = n_C, then

  D_0 = c_s
        + n_C * B_C * (K_C * c_s) / (1 + K_C * c_s)
        + φ_F * λ * W_s * (K_T * c_s) / (1 + K_T * c_s)

Interpretation:
  W_s is the sample-level weighted tumor DNA burden driving dye depletion, and φ_F modulates how strongly that burden is converted into effective accessible binding burden.

------------------------------------------------------------------
5. Further consolidation of constants
------------------------------------------------------------------

Define

  A_C := n_C * B_C
  A_T := λ

Then

  D_0 = c_s
        + A_C * (K_C * c_s) / (1 + K_C * c_s)
        + φ_F * A_T * W_s * (K_T * c_s) / (1 + K_T * c_s)

This preserves the same mechanistic structure as before, but now keeps φ_F explicit rather than absorbing it into A_T.

------------------------------------------------------------------
6. Non-dimensionalization
------------------------------------------------------------------

Absolute dye concentration and binding constants are not separately well identified, so rescale the system relative to the CEN binding parameter.

Define

  u_s := K_C * c_s
  ρ   := K_T / K_C

Then

  (K_C * c_s) / (1 + K_C * c_s) = u_s / (1 + u_s)

and

  (K_T * c_s) / (1 + K_T * c_s) = (ρ * u_s) / (1 + ρ * u_s)

Multiply the dye-conservation equation by K_C and define

  D_tilde   := D_0 * K_C
  A_C_tilde := A_C * K_C
  A_T_tilde := A_T * K_C

Then the mass-balance equation becomes

  D_tilde = u_s
            + A_C_tilde * u_s / (1 + u_s)
            + φ_F * A_T_tilde * W_s * (ρ * u_s) / (1 + ρ * u_s)

Interpretation:
  u_s is the non-dimensional free-dye state for sample s.
  ρ compares tumor and CEN binding-response curves.
  φ_F explicitly modulates the tumor contribution to dye depletion.

------------------------------------------------------------------
7. Final recommended reduced mechanistic model
------------------------------------------------------------------

For each sample s, solve u_s implicitly from

  D_tilde = u_s
            + A_C_tilde * u_s / (1 + u_s)
            + φ_F * A_T_tilde * W_s * (ρ * u_s) / (1 + ρ * u_s)

Then model the peak means as follows.

CEN peak mean:

  μ_C,s = M_C * u_s / (1 + u_s)

where

  M_C := a_C * B_C * K_C

Tumor peak mean:

  μ_m,s = μ_C,s * φ_F * R*_m,inf * (1 + u_s) / (1 + ρ * u_s)

Equivalent ratio form:

  μ_m,s / μ_C,s = φ_F * R*_m,inf * (1 + u_s) / (1 + ρ * u_s)

Interpretation:
  1. Increasing W_s lowers free dye through the mass-balance equation.
  2. Lower u_s shifts the CEN peak downward.
  3. If ρ != 1, tumor:CEN spacing changes with load.
  4. Even if ρ = 1, tumor:CEN spacing still depends on φ_F.

Special case:
  If ρ = 1, then

    μ_m,s / μ_C,s = φ_F * R*_m,inf

  and depletion is purely common-mode, but the tumor:CEN ratio still carries the fixation accessibility factor.

------------------------------------------------------------------
8. What this model captures
------------------------------------------------------------------

This reduced model captures all of the following:

  (i)   a global peak shift through μ_C,s
  (ii)  a differential tumor-vs-CEN compression/expansion through ρ
  (iii) a common tumor-side fixation accessibility effect through φ_F

Therefore it is the natural mechanistic form if one believes that:
  - finite dye availability affects all peaks,
  - tumor and CEN may respond differently to declining free dye,
  - and fixation changes tumor accessibility in a way that should remain explicit rather than being absorbed into nuisance constants.

------------------------------------------------------------------
9. What is safe to simplify, and what is not
------------------------------------------------------------------

Safe simplifications:
  - collecting CEN constants into M_C
  - defining R*_m,inf as the asymptotic tumor:CEN ratio excluding the common fixation factor
  - replacing separate tumor K_m with one common K_T (if accepted as a model assumption)
  - modeling tumor binding burden through W_s
  - rescaling c_s to u_s

Not safe without additional justification:
  - low-occupancy linearization: (Kc)/(1+Kc) ~ Kc
  - high-dye saturation approximation: (Kc)/(1+Kc) ~ 1
  - forcing ρ = 1 unless the data support common-mode depletion only
  - absorbing φ_F into A_T or into R_m,inf if one wants fixation effects to remain explicit and distinguishable in the symbolic model

------------------------------------------------------------------
10. Practical note on fitting
------------------------------------------------------------------

The most directly learnable quantities from peak-location data are typically combinations such as:
  - M_C
  - R*_m,inf
  - ρ
  - φ_F * A_T_tilde
  - φ_F * R*_m,inf

So although φ_F is now explicit in the model, it may still be only weakly identifiable separately from other scale parameters unless there is additional structure or external information. Still, if the goal is to preserve the formal influence of fixation—especially when the CEN spike and tumor material may not have shared the same fixation history—this is the correct reduced form to carry forward.
